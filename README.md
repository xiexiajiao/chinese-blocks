## 📝 使用示例

### 示例 1：让小猫移动
1. 从左侧找到 **🟡 事件**，拖入 `🚩 当绿旗被点击`
2. 在下方拼接 **🔵 运动** 里的 `📍 移动 10 步`
3. 点击 `▶️ 运行`

### 示例 2：画正方形
```
🚩 当绿旗被点击
  └─ 🔁 重复 4 次
      ├─ 📍 移动 50 步
      └─ 🔃 顺时针旋转 90 度
```

### 示例 3：打招呼
```
🚩 当绿旗被点击
  ├─ 💬 说 "你好！"
  ├─ ⏱️ 等待 1 秒
  └─ 💭 想 "让我想想..."
```

## 🛠️ 技术栈

- **Blockly** - Google 开源图形化编程库
- **Canvas 2D** - 绘制角色和动画
- **原生 JavaScript** - 无框架依赖，轻量快速
- **LocalStorage** - 本地存储项目

## 🔧 故障排除

### 积木无法拖拽
- 确保使用鼠标左键拖拽
- 检查浏览器是否支持 Blockly（推荐 Chrome/Firefox/Safari）

### 程序无法运行
- 确保添加了 `🚩 当绿旗被点击` 积木
- 检查积木是否正确连接（会有"咔哒"声）

### 角色不显示
- 检查角色大小是否为 0
- 确认角色未被隐藏（右下角开关）

### 项目无法保存
- 检查浏览器是否允许 LocalStorage
- 清除浏览器缓存后重试
