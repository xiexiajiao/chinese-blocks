# ✈️ 趣味飞行编程

一个类似 Scratch 的全中文图形化编程工具，专为儿童设计，以**飞机特技表演**为主题，让编程像玩游戏一样有趣！

### 方法 1：直接用浏览器打开
1. 找到 `index.html` 文件
2. 双击或用浏览器打开


### 积木分类（7 大类）

| 分类 | 颜色 | 功能 | 示例 |
|------|------|------|------|
| 🟡 事件 | 黄色 | 程序触发器 | 绿旗点击、飞机被点击、按键 |
| ✈️ 飞行控制 | 蓝色 | 基础飞行动作 | 向前飞、左右转弯、爬升、俯冲 |
| 🎭 飞行动作 | 深蓝 | 特技动作 | 桶滚、后空翻、前空翻、悬停 |
| ✨ 特效 | 紫色 | 视觉效果 | 烟雾、火焰、彩虹尾迹 |
| 🟠 控制 | 橙色 | 流程控制 | 等待、循环、条件判断 |
| 🔢 数字 | 绿色 | 数学运算 | 数字、算术、随机数、比较 |
| 📊 传感器 | 青色 | 飞行数据 | 高度、速度、方向 |

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+N` | 新建项目 |
| `Ctrl+S` | 保存项目 |
| `Ctrl+O` | 打开项目 |
| `Ctrl+Enter` | 运行程序 |
| `Escape` | 停止程序/关闭弹窗 |
| `↑↓←→` | 方向控制（需配合事件积木） |
| `F` | 加速 |
| `S` | 减速 |

## 📝 使用示例

### 示例 1：基础飞行
```
🚩 当绿旗被点击
  ├─ ✈️ 向前飞行 10 米
  ├─ 🔃 向右转弯 15 度
  └─ ✈️ 向前飞行 10 米
```

### 示例 2：桶滚表演
```
🚩 当绿旗被点击
  ├─ ✈️ 向前飞行 20 米
  ├─ 🌀 桶滚机动
  ├─ ✨ 喷射烟雾
  └─ 💬 说 "看我的特技！"
```

### 示例 3：连续特技
```
🚩 当绿旗被点击
  ├─ ✨ 彩虹尾迹
  └─ 🔁 重复 3 次
      ├─ ⭕ 后空翻
      ├─ 🔁 前空翻
      └─ ⏱️ 等待 0.5 秒
```

### 示例 4：键盘控制飞机
```
⌨️ 当按下 ↑ 上箭头
  └─ 📈 爬升 10 米

⌨️ 当按下 ← 左箭头
  └─ 🔄 向左转弯 15 度

⌨️ 当按下 F 加速
  └─ 💨 加速飞行
```

## 🛠️ 技术栈

- **Blockly** - Google 开源图形化编程库
- **Canvas 2D** - 绘制飞机和动画
- **原生 JavaScript** - 无框架依赖
- **LocalStorage** - 本地存储项目

1. 欢迎加入飞行学院

## 💾 项目管理

### 保存项目
1. 点击 `💾 保存`
2. 输入项目名称（如"我的首飞表演"）
3. 确认保存

### 加载项目
1. 点击 `📂 打开`
2. 从列表选择项目
3. 点击加载

### 删除项目
在打开列表中点击 `×` 按钮

## 🎨 飞机特色

- **5 种涂装**：蓝色、红色、绿色、橙色、紫色
- **3D 效果**：阴影和渐变模拟立体感
- **动态火焰**：加速时尾部喷射火焰
- **飞行尾迹**：留下彩色轨迹
- **粒子特效**：烟雾、火焰、彩虹粒子

## 🔧 故障排除

### 积木无法拖拽
确保使用鼠标左键，推荐 Chrome/Firefox/Safari

### 程序无法运行
确保添加了 `🚩 当绿旗被点击` 积木

### 飞机不显示
检查右下角开关是否打开
